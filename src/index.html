Mis Proyectos Destacados

Aplicación Web para el Metro de Medellín (Spring Boot)

Este proyecto es una innovadora aplicación web desarrollada con Spring Boot, diseñada para revolucionar la experiencia de los usuarios del sistema de transporte Metro de Medellín. Nuestro objetivo principal es optimizar los viajes al proporcionar información más precisa, clara y en tiempo real. Esto incluye detalles sobre rutas, horarios estimados, y posibles interrupciones.

Además de mejorar la planificación de los viajes, la aplicación integra una funcionalidad crucial para la comunidad: permite a los usuarios reportar cualquier problema o incidencia que observen dentro de las instalaciones o el servicio del metro. Desde averías en escaleras eléctricas, aglomeraciones, hasta situaciones que requieran atención inmediata, esta herramienta busca fomentar la participación ciudadana y contribuir a un servicio más eficiente y seguro para todos. Desarrollado con tecnologías robustas de backend y pensando en la usabilidad, este proyecto es un claro ejemplo de mi capacidad para construir soluciones completas y funcionales.
